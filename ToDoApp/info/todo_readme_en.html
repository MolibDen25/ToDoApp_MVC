Summary

Console app for managing ToDo notes using an MVC structure. Data are stored in the notes folder as .txt files. The app supports crossing-out, editing and deleting notes and lines.

Project structure

ToDoApp/
├── Controllers/    # Menu and action handling logic
├── info/           # README folder
├── Views/          # Responsible for console display
├── Program.cs      # Entry point
└── notes/          # Saved notes (created automatically)

Run

1. Install .NET SDK (6.0+). Check in terminal with:

dotnet --version

2. On Windows using a shortcut ToDoApp or run_todo.bat.

Or in terminal:

cd ToDoApp
dotnet run

Controls

Main keys and commands:

  • 1 — select item / note
  • + — add a new note / line
  • ~№ — cross/un-cross note or line
  • -№ — delete note or line
  • Enter — save edit
  • Esc — cancel edit
